## ✨ Features

### Public Portfolio
- **Home** — Hero with typewriter, bio, skills, experience timeline, education
- **Projects** — 3 real projects (Nail Disease Detection, Movie Recommender, Student Management System)
- **Blog** — Search, filter (Published/Draft/All), skeleton loaders
- **Single Blog** — Markdown rendering, reading time, tags

### Admin Portal
- **Login** — `admin` / `admin123`
- **Dashboard** — Stats (total, published, drafts)
- **Blog CRUD** — Create, Edit, Delete, Toggle status

### UX
- Dark / Light mode
- Toast notifications
- Fully responsive
- Smooth animations
